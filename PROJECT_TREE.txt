QUICK START
===========

1. Start Backend:
   cd backend
   pip install -r requirements.txt
   python main.py

2. Start Frontend:
   cd frontend
   npm install
   npm start

3. Play:
   Open http://localhost:3000
   Use arrow keys to move
   Avoid the red police agent!


API ENDPOINTS
=============

Health:
  GET  /                    Service status

Game:
  POST /record_move         Record player movement
  POST /ai_strategy         Get AI decision
  POST /game_state          Update game state

Analytics:
  GET  /statistics          AI statistics
  GET  /patterns            Pattern analysis
  POST /predict_move        Predict next move

Control:
  POST /reset               Reset game
  POST /reset_learning      Clear learned data
